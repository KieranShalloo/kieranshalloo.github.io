Trainline

The Google Problem

Brief

To create ways to reassure passengers coming to Trainline from Google, communicating the positive propositions of the product.

Client

Trainline

Project Date

2024

Role

Senior UX/UI Designer (contractor)

Google Trains

Google has introduced a Google train module, which allows users to search for their journeys on google rather than using the trainline service. Whilst good for the user the integration with the trainline allows users to go directly to the train search result without seeing any of the user benefits of using the Trainline platform.

Approaching the problem

With a view to get this feature live as fast as possible we had a 5 day turn around to go from a concept to a design that was ready for development. I took the initiative to bring some elements of existing designs and concepts already in the design system to bring this to life. With one eye on the developer resource needed for this project it was all about quick wins and a scalable concept we could iterate on top off!

Iteration 1

Version 1

The quick win for this project would be to reuse some design system principles but to adjust them for the user coming from the Google Trains platform. This, for me, meant that we should value the journey the user has made to come to the trainline above everything else.

I first brought the selected train journey to the top of the page. Clear messaging is to be applied to the selected option to make it stand out as clearly as possible. Underneath a reassurance message will be placed in a easy to access position giving the user the benefits of using the Trainline platform.

This clickable element will open a new modal which will be able to sell personalised benefits of the platform based on common highly rated features with positive feedback.

Iteration 2

Version 2

With the second iteration of the design, the goal was to put a lot more emphasis on highlighting the Trainline features, especially in european markets where the product doesnt have the brand recognition it does in the UK
